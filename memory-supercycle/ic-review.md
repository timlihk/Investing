# Memory Supercycle IC Review

Date: 2026-03-25

Scope: Independent committee review of the current `memory-supercycle` project. This memo does not restate the scorecard. It tests whether each name is a real bottleneck, a good but cyclical beneficiary, or a weaker expression of the theme.

## Committee View

- The core thesis is credible: AI has made HBM, advanced packaging, probe, and test materially tighter than prior memory cycles.
- The best ideas are not all memory manufacturers. The strongest underwriting cases are the narrow, qualified choke points where replacement is slow and the tool or consumable is a tiny share of system cost.
- Several names in the folder are really the same macro bet: AI server and accelerator capex must stay elevated through 2026-2027. If that pauses, correlations across the basket will be much higher than the individual writeups suggest.
- The weakest part of the folder is storage/NAND. Those names can work, but they rely more on cycle discipline than on true chokepoint economics.

## Company Review

### 000660.KS - SK hynix
Repo stance: `BUY` (88)

IC take: This is still the best single-stock expression of the memory supercycle. The thesis is not just "DRAM prices up"; it is "HBM is the scarce layer and SK hynix is leading it operationally." That said, this is no longer an early discovery. A lot of the rerating has already happened, so the remaining upside depends on sustaining leadership rather than merely participating.

Top risks / disconfirming evidence: HBM4 share loss, Samsung closing the quality/yield gap faster than expected, or evidence that current margins are peak-cycle rather than structurally higher.

### MU - Micron Technology
Repo stance: `BUY` (86)

IC take: I agree with inclusion and broadly agree with the ranking. Micron is the cleanest U.S.-listed memory name and a real beneficiary of HBM plus AI SSD demand. The independent question is whether this deserves to be treated like a choke point or simply a better memory cycle. I lean toward the latter: attractive, but still one step below SK hynix in franchise quality.

Top risks / disconfirming evidence: HBM execution lag versus SK hynix, aggressive capex into a cyclical market, and a faster normalization in DRAM/NAND pricing once supply responds.

### 042700.KS - Hanmi Semiconductor
Repo stance: `BUY` (83)

IC take: The bottleneck logic is strong. If TC bonders remain a narrow, high-qualification market, Hanmi deserves a premium. But this is the type of name that can look like a monopoly right until a customer dual-sources or a competitor clears qualification. I would treat it as high-upside, small-size exposure rather than a core holding.

Top risks / disconfirming evidence: customer concentration, Hanwha or others winning meaningful share, order lumpiness, and any evidence that HBM process flows are becoming less dependent on Hanmi's specific tool position.

### 0522.HK - ASMPT
Repo stance: `BUY` (81)

IC take: This is a credible underfollowed packaging name and one of the more interesting ideas in the folder. I like the installed-base argument more than the current margin profile. The key committee issue is purity: ASMPT is not only an HBM story, so the stock may not get paid like a direct bottleneck even if the underlying TCB franchise is sound.

Top risks / disconfirming evidence: broader business mix diluting the thesis, slower conversion from HBM order wins into earnings, and a market that continues to value ASMPT as a diversified equipment vendor.

### 8035.T - Tokyo Electron
Repo stance: `BUY` (80)

IC take: This is probably the safest high-quality name in the folder even if it is not the most exciting. The process-of-record exposure across DRAM, HBM thinning, and packaging is exactly what a committee should want when the thematic call is right but timing is uncertain. The objection is valuation style, not business quality: this is a recognized large-cap, not a hidden idea.

Top risks / disconfirming evidence: front-end wafer fab equipment digestion after a heavy AI build cycle, China weakness, and the possibility that memory capex stays healthy while broader semi-cap sentiment still compresses the multiple.

### FORM - FormFactor
Repo stance: `BUY` (79)

IC take: One of the better undercovered names here. Probe cards are a good example of a qualified consumable that becomes more valuable as HBM complexity rises. I support the thesis, but I would still want proof that HBM growth converts into durable margin expansion rather than simply higher but still lumpy revenue.

Top risks / disconfirming evidence: margin pressure during ramps, customer inventory swings, and evidence that probe-card competition is stronger than the reports imply.

### SIMO - Silicon Motion
Repo stance: `BUY` (78)

IC take: This is a sensible storage-enabler name, but I am less convinced it belongs in the same tier as the true memory choke points. Merchant controllers matter, especially in outsourced SSD stacks, yet the insourcing ceiling is real. I would keep it in the basket, but I would not underwrite it as a core supercycle bottleneck.

Top risks / disconfirming evidence: controller insourcing by major NAND vendors, slower enterprise ramp for MonTitan-type products, and the possibility that a large part of demand remains tied to ordinary client/mobile recovery rather than AI storage.

### SNDK - Sandisk
Repo stance: `BUY` (77)

IC take: This is where I diverge most from the repo stance. Sandisk may work as a flash-cycle recovery with AI storage upside, but it does not look like a chokepoint. The High Bandwidth Flash angle is interesting, not investable fact. I would view this as a tactical NAND/storage exposure rather than a core committee pick.

Top risks / disconfirming evidence: flash oversupply returning, enterprise SSD demand normalizing, HBF staying conceptual, and the stock already discounting a large amount of good news after separation.

### 6857.T - Advantest
Repo stance: `BUY` (76)

IC take: High-quality and defensible. Test intensity usually increases when devices become harder to ship, and HBM clearly fits that pattern. I like this better than some of the lower-quality pure-play memory beneficiaries because it monetizes complexity without needing memory pricing to remain euphoric. It is not pure memory, but that is a feature, not a bug.

Top risks / disconfirming evidence: pull-forward demand creating a digestion phase, non-memory SoC weakness obscuring the thesis, and the usual capital-equipment cyclicality after a sharp AI-led order burst.

### 6315.T - TOWA
Repo stance: `BUY` (75)

IC take: Reasonable to include, but speculative. Current order momentum looks real, yet a meaningful part of the bull case rests on future HBM4 or panel-level packaging relevance that has not been proven economically. I would not pay up for roadmap optionality unless there is clearer evidence that customers are standardizing around TOWA's approach.

Top risks / disconfirming evidence: roadmap slippage, customers choosing other packaging schemes, and back-end order strength fading before the next technology node translates into revenue.

### 285A.T - Kioxia Holdings
Repo stance: `HOLD` (74)

IC take: The current stance is appropriate. Kioxia is a legitimate NAND and enterprise SSD beneficiary, but the business is still more cyclical than scarce. From a committee perspective, it is also hard to separate this idea from Sandisk because the economics overlap heavily through the joint venture.

Top risks / disconfirming evidence: NAND pricing mean reversion, weak enterprise mix, rising capex without durable return improvement, and duplication of exposure if Sandisk is also owned.

### AMKR - Amkor Technology
Repo stance: `HOLD` (74)

IC take: Strategically important, financially less attractive. I buy the argument that advanced packaging capacity is valuable, especially with U.S. expansion, but OSAT remains a structurally tougher business than the clean tooling and consumables bottlenecks. This is investable, though mostly as a second-order beneficiary rather than a top expression of the theme.

Top risks / disconfirming evidence: thin margins staying thin, customers keeping the best packaging economics in-house, and Arizona taking longer to produce acceptable returns than the strategic narrative suggests.

### 8110.TW - Walton Advanced Engineering
Repo stance: `HOLD` (72)

IC take: I understand why it is in the file set, but it is not yet a committee-quality idea. The specialization is real, yet the latest operating numbers are weak enough that the thesis still depends on an earnings recovery that has not occurred. In other words, the franchise may be better than the P&L, but the P&L still matters.

Top risks / disconfirming evidence: persistent negative operating profitability, customer concentration, non-operating gains masking core weakness, and the possibility that niche memory backend work remains structurally lower return than hoped.

### 005930.KS - Samsung Electronics
Repo stance: `HOLD` (71)

IC take: Correctly included, correctly discounted. Samsung is too important to ignore in any memory review, but it is not the best way to express this theme today. The committee should not pay for theoretical HBM leadership that the market has not yet seen in practice. If Samsung proves it can close the HBM gap, upside exists; until then, it is a broad quality conglomerate with a memory call embedded inside it.

Top risks / disconfirming evidence: continued HBM execution slippage, the conglomerate discount persisting, and non-memory divisions diluting what would otherwise be a strong memory-cycle setup.

## Bottom Line

- Names I would be most comfortable underwriting as core positions: `000660.KS`, `MU`, `8035.T`, `FORM`, `6857.T`.
- Names I would own only as higher-risk satellites: `042700.KS`, `0522.HK`, `6315.T`, `SIMO`.
- Names I would treat as cyclical or lower-quality expressions of the theme rather than true choke points: `SNDK`, `285A.T`, `AMKR`, `8110.TW`, `005930.KS`.
- The main thing that would invalidate much of this folder at once is simple: AI memory demand must stay supply-constrained longer than the market expects. If that premise weakens, the bottlenecks will still be the best holdings, but the storage and lower-margin service names could rerate down quickly.

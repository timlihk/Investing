# Memory Supercycle Research

Research session: 2026-03-24

## Thesis
AI is turning memory from a cyclical commodity into a system bottleneck. The highest-conviction names are no longer just the biggest DRAM or NAND vendors, but the companies controlling scarce HBM supply, HBM packaging, memory test, probe, and AI-linked storage layers.
